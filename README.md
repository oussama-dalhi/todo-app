# 📝 Todo App

A simple and clean Todo App to manage your daily tasks.

## 🔗 Live Demo

👉 https://oussama-dalhi.github.io/todo-app/

## ✨ Features

* Add and delete tasks
* Mark tasks as completed
* Persistent storage (saved in browser)
* Dark / Light mode toggle

## 🛠️ Built With

* HTML
* CSS
* JavaScript

## 🚀 How to Use

1. Enter a task
2. Click "Add"
3. Check tasks when done
4. Delete anytime
